# Hazelcast Test

## Hazelcast 名字由来

1. **“Hazel”** 是榛树。在欧洲文化里，榛树被视为**智慧与灵感的象征**。
2. **“Cast”** 一词有多重含义：
    * 技术上，它让人想到 **“broadcast”**（广播）或 **“multicast”**（组播）——这正是 Hazelcast 的核心机制之一：节点间通过网络广播自动发现、同步数据。
    * 英语中 “cast” 也有 “投射、分布” 的意思，对应 Hazelcast 的定位——**分布式缓存与计算平台**。
3. 传达出分布式（cast）与智能（hazel）的理念。

## 模块说明

- hazelcast-embed 嵌入测试
- hazelcast-alone-community 社区版独立服务测试
- hazelcast-alone-enterprise 企业版独立服务测试
