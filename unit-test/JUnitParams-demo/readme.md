## 参数化测试

### 参考
- 参考类：https://github.com/Pragmatists/junitparams/blob/master/src/test/java/junitparams/usage/SamplesOfUsageTest.java
- 与 Spring 集成参考：https://github.com/Pragmatists/junitparams-spring-integration-example

### 注意点
- 1. 通过方法或类传递参数时，要运行整个测试类
- 2. Eclipse 下运行，在 `@Test` 处按 `F11`，不要在参数提供类里面按 `F11`
